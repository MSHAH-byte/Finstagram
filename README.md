# Finstagram

Finstagram is a full-stack social media application that goes beyond a simple UI clone. It features a complete ecosystem for media sharing, personalized user accounts, and real-time social interactions.

## Overview
This project focuses on complex data relationships within **Firebase**. It allows users to manage their own digital identity through personal profile galleries while engaging with a global community through interactive feeds.

## Technical Highlights
* **Personalized User Profiles:** Implemented logic to filter and display user-specific media, ensuring that each user has a dedicated gallery for their uploads.
* **Social Engagement Suite:** Developed interactive systems for **Likes, Comments, and Sharing**, utilizing real-time Firestore listeners.
* **Firebase Cloud Storage:** Optimized image upload workflows, handling high-resolution media and generating secure download URLs.
* **Relational NoSQL Design:** Engineered a Firestore schema that efficiently links posts to individual users while allowing global feed access.
* **Media Picker Integration:** Seamless integration of camera and gallery access for instant content creation.

## Tech Stack
* **Framework:** Flutter (Dart)
* **Backend:** Firebase (Authentication, Firestore, Cloud Storage)
* **Social Logic:** Real-time data syncing and interaction tracking.
